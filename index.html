Lógica de Programação

O que é a lógica de programação?

A lógica de programação é o uso da lógica para criação de um código, ou seja, realizar o passo a passo de uma ação que será executada pelo código. Esta ferramenta é muito importante pois é através do entendimento dela que se torna possível criação e formatação de um código funcional e responsivo. Mas como que podemos compreender a lógica por detrás dos programas, ou melhor, dos computadores?

Para isso podemos pensar que os primeiros computadores foram criados para serem utilizados como calculadoras. Sim, e essa base voltada para os cálculos está incorporada nos computadores atuais. Mas hoje em dia os computadores fazem muitas funções além de cálculos, eles conseguem realizar escolhas situacionais, cálculos extremamente rápidos e complexos, traduzir imagens e sons para códigos e vice versa. E tudo isso só é possível devido à códigos responsívos e complexos que utilizam muita lógica de programação para funcionarem em conjunto.

    Tipos de dados

    Na programação podemos utilizar diferentes tipos de dados. A variável acima que vimos utiliza dados numéricos, mas em totalidade existem outros tipos de dados. Vamos ver alguns deles:

  • Dados booleanos: os dados booleanos sempre retornarão uma afirmativa, ou seja, dirão se um determinado intervalo é verdadeiro (true) ou falso (false). A utilização dos dados booleanos nos permite executar determinada ações a depender da veracidade de um intervalo, ou seja, permite que o programa tome uma decisão dependendo das ações, ou condições, precedentes. E isto é extremamente interessante para a programação pois possibilita a criação de um código mais limpo mesmo que tenha uma complexidade grande.
  • Números inteiros: Estes são os números sem fração, como por exemplo 1, 2, 3, 4...
  • Números fracionários: São constituídos dos números acrescids de ponto, como por exemplo 1.2, 1.1, 1.3, 3.1, ...
  • Texto: E o texto que é constituído por todo e qualquer texto que você queira utilizar como dado.

    Tornando o programa inteligente

    Podemos tornar nossos programas mais inteligentes ao utilizar do recurso de desvio de fluxo. Isto é possível devido aos operadores. Imagine um estrada reta, o seu programa é o veículo que vai percorrer esta estrada. Quando adicionamos desvios de fluxos é como se ramificassemos está estrada possibilitando que o veículo percorra diferentes caminhos para o mesmo destino ou para destinos diferentes. E este é o pulo do gato, pois possibilitando que exista a escolha de caminhos é possível tornar o programa mais inteligente, uma vez que ele será capaz de decidir qual caminho tomar a depender da necessidade momentânea. E como isto é realizado na programação?
    Na programação é possível criar desvios de fluxo com o uso do "IF" ou "SE" em português. Ao utilizarmos o IF o nosso programa irá tomar a decisão baseado em algo, por exemplo: "se a tela estiver ligada, faça X coisa. Senão faça Y coisa."
    Observe que o IF só executará a condição X se a condição "tela ligada" for verdadeira. Do contrário o programa irá executar outra condição programada. Veja um exemplo de uma parte de um programa que permite que apenas quem possua uma característica consiga ter acesso a página:

    var caracteristica_necessaria = X
    var caracteristica_da_pessoa = Y

    if (caracteristica_da_pessoa >= caracteristica_necessaria) {
    alert ("acesso autorizado" + permite acesso)
    } else {
    alert ("acesso negado" + fecha página) }

    Note que utilizamos variáveis para incluir os atributos que iremos mensurar, utilizamos também o "else" e o "alert". O "else" é o nosso "senão", ou seja, é o código que faz com que nosso algorítimo siga por outra via caso a condição inicial não seja satisfeita. E o "alert" é o código que exibirá um alerta no navegador para avisar a pessoa que está acessando a página o que foi decidido de acordo com sua resposta e os pré-requisitos da própria página. E a partir deste simples exemplo podemos dar uma infinidade de funções para um código, além disso podemos utilizar o IF e ELSE de formas muito mais complexas que estas, coisa que veremos nos próximos passos.

    Else if

    O else if é uma forma de classificar o if dentro de outra tomada de decisão anterior. Observe:

    var 1 = variavel_definida_pelo_usuário

    if (1 == "X") {
    alert("A")
    } else if (1 == "Y") {
    alert ("B")
    } else (1 == "Z"){
    alert ("C")
    }

    E qual é a diferença entre o else if e o else?
    O IF classificado (else if) serve para determinar uma ação de acordo com a veracidade de uma característica específica, no caso do nosso exemplo ele só seria executado caso a primeira característica fosse falsa e a segunda verdadeira. Senão ele cairia na terceira característica (else). De maneira simples a diferença é que o else if só é executado quando cumpre a característica atribuída a ele, enquanto o else é executado sempre que não atender as características anteriores, sejam elas do if ou else if.

    Para criar códigos ainda mais complexos podemos ainda utilizar os comparadores para que o programa fique ainda mais versátil. Veja um exemplo:

    var 1 = prompt ("Digite a variante 1 aqui")
    var 2 = prompt ("digite a variante 2 aqui")

    if (1 == "X" && (2 == "XX")) {
    alert("A")
    } else if (1 == "Y" && (2 == "YY" || 2 == "YX")) {
    alert ("BY")
    } else if (1 == "Z" && (2 == "ZZ")){
    alert ("CZ")
    } else{
    alert ("Var não reconhecida")
    }

    Entendendo o código:
    o comando var é onde iremos inserir nossas variáveis, é o que faz com que o código seja mais amplo (variáveis alteráveis) ou mais fechado (variáveis fixas). Neste exemplo estamos utilizando o prompt para permitir que as variáveis sejam alteráveis. O if por sua vez é o que vai possibilitar a execução do alert("A") se o seu conteúdo for verdadeiro, da mesma forma os else if irão atribuir alertas quando sua condição for verdadeira, caso contrário o código irá continuar até cair no else, que irá realizar o alert("Var não reconhecida"). Vale lembrar que o else representa todo o restante que foge às condições verdadeiras.

    Repetições

    As repetições são blocos do código que podemos programar para executar um loop até que uma determinada condição seja alcançada. Este é outro recurso riquíssemo que faz com que o seu programa ganhe muito mais vida e possibilite que você atribua características ao seu código que o tornam extremamente mais versátil se utilizado da maneira correta e em conjunto com outros comandos como o "else if" por exemplo. E como podemos realizar esse loop?

    Para isso podemos utilizar um comando ue representa o "enquanto", este comando é o while. Veja um exemplo de loop utilizando o while:

    var contador = 1

    while (contador <=10000) {
    console.log(contador + "Contando")
    contador = contador + 1
    }
    console.log("Fim da Contagem")

    Este programa é um contador que utiliza a função while para executar a contagem de 1 + 1 até chegar no número 10000. Parece um número alto certo? Mas vamos nos atentar que estamos criando um contador simples que somente irá executar a conta até atingir o alvo e que somente é possível visualizar através do console, e devemos considerar também a velocidade de processamento dos computadores atuais, que é elevada e faz com que o número seja alcançado em alguns segundos. Observe que este contador não mensura tempo ou qualquer outra variável, para isso teríamos que utilizar elementos do JavaScript e CSS, conteúdos que teremos posteriormente.
    Também é possível utilizar o comando "alert" ou outros comandos que demonstrem na tela a contagem. Para a utilização do alert basta substituir o "console.log" pelo comando "alert" no código. Mas neste caso você terá que clicar em ok a cada número da contagem.

Então podemos ver que as funções possuem diversas funções, além disso vale ressaltar que as funções atum como um desvio de fluxo para um local pré determinado.
